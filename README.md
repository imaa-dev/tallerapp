# 🔧 TallerAdmin

Sistema de gestión de servicios para talleres técnicos y maestros reparadores.

Permite registrar, organizar y documentar cada servicio realizado, manteniendo un historial completo del proceso de reparación mediante imágenes, reportes, reseñas y puntuaciones del servicio.

---

## 📋 Descripción

TallerAdmin es una plataforma diseñada para ayudar a talleres y técnicos independientes a mantener un control cronológico de los trabajos realizados.

Cada servicio puede ser documentado desde su ingreso hasta su entrega, permitiendo registrar información relevante, evidencias fotográficas, observaciones técnicas y avances del proceso.

---

## ✨ Características

### 🗂 Gestión de Servicios

* Registro de nuevos servicios.
* Seguimiento cronológico de cada reparación.
* Historial completo de modificaciones.
* Estado actual del servicio en tiempo real.
* Envio de reportes por fase de reparación al cliente

### 📸 Evidencia Fotográfica

* Carga de imágenes asociadas al servicio.
* Registro visual antes, durante y después de la reparación.
* Organización de evidencias por etapas del trabajo.

### 📝 Reportes de Avance

* Creación de reportes durante el proceso de reparación.
* Registro de observaciones técnicas.
* Seguimiento detallado de actividades realizadas.

### ⭐ Reseñas y Evaluaciones

* Registro de comentarios sobre cada servicio.
* Historial de observaciones y resultados.
* Información útil para futuras intervenciones.

### 📊 Reportes y Estadísticas

* Reportes de servicios completados.
* Seguimiento de trabajos en proceso.
* Información histórica para análisis y control operativo.

---

## 🚀 Beneficios

* Mantiene el historial completo de cada trabajo.
* Mejora la organización interna del taller.
* Facilita la comunicación con clientes.
* Permite respaldar reparaciones mediante evidencia fotográfica.
* Centraliza toda la información del servicio en un único lugar.
